 Projeto: Cálculo da Média de 4 Notas 

Este é um projeto simples em Java que realiza a leitura de 4 notas inseridas pelo usuário, calcula a média e imprime o resultado no console.

 Tecnologias utilizadas

 Java 11
 Maven
 IntelliJ IDEA

 Estrutura

 `src/tarefa/calculo/CalculoMedia.java`: Código principal com leitura das notas e cálculo da média.
 `pom.xml`: Configuração Maven para compilação com Java 11.
